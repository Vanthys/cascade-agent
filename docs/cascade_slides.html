Our Contribution

One workflow: literature → graph → hypothesis

01 — Real-time graph

Type any gene symbol. Within seconds, Cascade fetches curated facts from MyGene.info, maps interactions from STRING DB and OmniPath — including correct activates / inhibits directionality — and renders an interactive force graph.

02 — Grounded summaries

Every node and edge comes with an AI-written summary that explicitly separates known facts (evidence-backed) from generated hypotheses. Clicking any node or edge opens an in-panel conversation with streaming responses.

03 — What-if hypotheses

Select a protein, choose a perturbation (inhibit, activate, up/downregulate), and run. The hypothesis agent propagates the effect through the full interaction graph and returns ranked downstream candidates with confidence scores and uncertainty notes.

Technology

Built on sponsor infrastructure

GMI Cloud

OpenAI-compatible inference API. Powers all four LLM calls — gene summaries, edge explanations, what-if hypotheses, and conversational follow-ups. Deterministic structured JSON output enforced via strict system prompt.

Kimi K2 (Moonshot AI)

Our LLM of choice. Long-context window lets us feed the full local subgraph, evidence snippets, and session history into a single prompt. Hosted on GMI Cloud for low-latency inference.

HydraDB

Persistent agent memory. Each session stores every gene explored, every hypothesis generated, and every conversation exchange. Recalled as semantic context on every new query — enabling true multi-turn research sessions.

Live Demo

How it works — step by step

①
Type a gene

Enter any gene symbol — BCL2, TP53, EGFR, BRCA1. Cascade resolves aliases automatically.

②
Graph appears live

Interaction network streams in via SSE. Nodes colour-coded by type; edges labelled activates / inhibits / binds.

③
Ask & hypothesise

Click any node to expand it, ask follow-up questions, or switch to What-if to simulate a perturbation.

Step 1 — Enter your gene of interest

Clean, focused input

  • No sign-up, no configuration required
  • Alias resolution — type "BCL-2" and get BCL2
  • Quick-access suggestion buttons for common genes
  • Capability tags hint at what the agent can do

Step 2 — AI researches the gene

Six-stage streaming pipeline

  • Symbol normalisation via MyGene.info
  • Gene facts: aliases, KEGG pathways, full summary
  • STRING DB + OmniPath interaction fetch
  • Graph patch events stream to the browser live
  • Kimi K2 summary generated while graph renders

Step 3 — Explore the interaction graph

Interactive force graph

  • Red arrows = inhibits; green = activates
  • Blue = binding; purple = coexpression / pathway
  • Right panel shows AI summary + evidence sources
  • Ask follow-up questions directly in the panel

Step 4 — Ask follow-up questions

Grounded conversational AI

  • Full session context carried across every message
  • Known facts clearly separated from AI hypotheses
  • Responses stream token-by-token in the panel
  • Evidence citations link back to source databases

Step 5 — Expand nodes on demand

On-demand graph expansion

  • Click any neighbor node to load its own interactions
  • New subgraph patches stream in and merge live
  • Panel updates with the selected node's summary
  • Graph layout re-simulates to accommodate new nodes

Step 6 — Set up a What-if perturbation

Four perturbation types

  • Inhibit — model drug blockade (e.g. venetoclax on BCL2)
  • Activate — simulate overexpression or agonist
  • Up / downregulate — continuous dose modelling
  • Click the graph to select any protein as target

Step 7 — Hypothesis + cascade overlay

AI-generated cascade analysis

  • Graph highlights propagated downstream effects in green
  • Target node marked BLOCK with red overlay
  • Panel shows: Known Context · Hypothesised Effects · Downstream Candidates
  • Confidence score + uncertainty notes included
